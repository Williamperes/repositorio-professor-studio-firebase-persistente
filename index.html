AulaStudio

Organizador de materiais, planos de aula, exercícios, projetos e cronogramas para o professor.

Versão Studio com edição de conteúdo, materiais prontos e conexão protegida.

Use sem configurar nada, salvando no navegador, ou conecte ao Firebase para sincronizar em nuvem. A conexão com o Firebase pode ser protegida por senha para evitar alterações indevidas nas configurações.

CRUD completoCrie, edite, visualize e exclua disciplinas e materiais.
Conteúdo pedagógicoCampos para objetivo, competências, carga horária, etapa e tags.
Materiais iniciaisCards prontos de banco de dados, front-end, back-end, APIs, Git e testes.
Firebase protegidoConecte ou desconecte o Firebase usando uma senha definida pelo professor.
Como deseja iniciar?

Para testar rapidamente, entre no modo local. Para usar em mais de um computador, configure o Firebase com uma senha de proteção.

Os dados ficarão salvos no navegador usando localStorage. Ideal para aula, teste e uso pessoal.
No primeiro acesso, essa senha será salva neste navegador. Depois, ela será exigida para conectar, alterar ou desconectar o Firebase.
